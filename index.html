My Projects

Project 1: Metro Assist

Metro Assist is a web-based application designed to help metro commuters retrieve lost belongings and receive station alerts using location tracking. The platform allows users to report or search for lost items and notifies them as they near their destination to prevent missing their stop. Tools Used: HTML, CSS, JavaScript, Firebase. Role: Developed the entire front-end and integrated Firebase for real-time data storage and retrieval.

Project 2: Personal Portfolio

This is a responsive and minimalistic personal website to showcase my skills, projects, and contact information. Designed with smooth user experience in mind, the site includes animated transitions, a project gallery, and a built-in contact form. Tools Used: HTML, CSS, JavaScript. Role: Designed and built the full website layout, styling, and interactive components.

Project 3: Journal App

A basic web journal application that lets users write, edit, and save personal entries. Data is stored locally or using a lightweight backend to maintain privacy and simplicity. Tools Used: Python (Flask), HTML, CSS, SQLite. Role: Built the backend architecture using Flask and implemented data handling and entry management.
